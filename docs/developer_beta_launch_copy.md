# OSCIRIS Developer Beta Launch Copy

## Full Announcement

OSCIRIS Developer Beta is now open.

Install the `osciris-node` CLI, generate your contributor identity, run the
local protocol demo, sync the public proof bundles, and join the early provider
and verifier workflow.

This beta is for developers and contributors who want to join the early
protocol workflow, inspect the public proof surface, and participate as
providers or verifiers during the current MVP stage.

## Call To Action

Download the latest `osciris-node` beta from GitHub Releases, then follow the
Quick Start in `protocol-rs/README.md`.
